Managed Self Custody wallet for AI Agents

holder is a self custody wallet allowing AI agents to use the wallet controlled by security policies. Spending limits are enforced by an account abstraction smart contract layer. Keys are held in browser memory and connected via Rust gRPC API to an external signer module with HSM. Supports accounts on Solana and EVM with an x402 payment client built in.

Web3 Game with AI Judge

A Web3 AI game where players send a small bet and a text to be judged by an LLM. LangGraph provides a quality AI result and verification. In the Rust Solana version, smart contracts are optimised to provide the ultimate on-chain provenance and verification of fairness, within minimal gas fee limits. Includes compressed NFTs and ZK compression of game history.
